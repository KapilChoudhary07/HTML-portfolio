# 🌐 Personal Portfolio – HTML Only

This is a simple **personal portfolio website built using pure HTML only**.  
The project is created for learning purposes and academic use, without using any CSS or JavaScript.

---

## 📌 Project Overview

This portfolio page displays basic personal and professional information, including skills, certificates, and contact details.  
All navigation is handled using **HTML anchor links**.

---

## ✨ Features

- ✅ Built using **only HTML**
- ✅ No CSS or JavaScript used
- ✅ Single-page layout
- ✅ Internal navigation using anchor tags
- ✅ Skills section
- ✅ Certificates section
- ✅ Contact section with clickable links
- ✅ LinkedIn and GitHub profile links

---

## 🛠️ Technology Used

- **HTML (HyperText Markup Language)**
